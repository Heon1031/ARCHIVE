# API 문서 인덱스

8단계 API 연결 전에 읽을 문서의 길찾기입니다. 필요한 범위의 문서만 읽고 작업합니다.

## 문서 목록

- `01_API_SCOPE.md`: MVP API 연결 범위와 프론트/백엔드 경계를 다룹니다.
- `02_TOKEN_SECURITY_POLICY.md`: Access Token, App Secret, 환경변수, UI 노출 금지 원칙을 다룹니다.
- `03_INSTAGRAM_API_PLAN.md`: Instagram API 연결 흐름과 정규화 원칙을 다룹니다.
- `04_THREADS_API_PLAN.md`: Threads API 연결 흐름과 지표 차이 대응 원칙을 다룹니다.
- `05_API_IMPLEMENTATION_STEPS.md`: 8단계 API 구현을 작은 하위 단계로 나눕니다.
- `06_API_FAILURE_POLICY.md`: API 실패 케이스와 복구/기록 원칙을 다룹니다.
