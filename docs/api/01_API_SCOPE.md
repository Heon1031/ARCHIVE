# API 연결 범위

## MVP에서 할 작업

- 등록된 Instagram / Threads 계정의 연결 상태를 확인한다.
- 계정별 최근 게시물 목록을 조회할 수 있는 구조를 만든다.
- 게시물별 인사이트를 조회해 `InsightRecord`로 저장할 수 있게 한다.
- API 응답은 UI에서 직접 쓰지 않고 내부 모델로 정규화한다.
- API 실패 시 앱이 멈추지 않고 사용자에게 다음 행동을 안내한다.

## MVP에서 하지 않을 작업

- 자동 업로드
- 상용 OAuth 전체 플로우
- 토큰 자동 갱신
- 로그인, 결제, 팀 협업
- 고급 AI 추천
- 서버 DB 기반 운영

## 프론트 단독으로 가능한 작업

- 계정 ID와 Access Token 입력값 관리
- API 연결 버튼과 결과 메시지 표시
- 정규화된 성과 데이터 표시
- localStorage 기반 개인용 테스트
- 실패 상태 안내와 수기 입력 보조 흐름 연결

## 백엔드가 필요한 작업

- App Secret 보관
- 토큰 교환, 갱신, 장기 토큰 관리
- 배포 환경에서의 안전한 API 프록시
- 사용자별 토큰 암호화 저장
- rate limit 제어와 서버 로그 관리

## 검증 원칙

실제 endpoint, metric, 권한 이름은 구현 단계에서 최신 공식 문서로 검증한다. 확실하지 않은 API 세부사항은 문서나 코드에서 단정하지 않는다.
