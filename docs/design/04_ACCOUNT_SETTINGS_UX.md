# 계정 등록·설정 탭 UX

## 목적

계정 등록·설정 탭은 Instagram / Threads 계정과 API 연결 상태를 관리하는 화면이다.

## 기본 영역

- 계정 목록
- Instagram 계정 등록
- Threads 계정 등록
- 계정 ID 입력
- Access Token 입력 또는 관리 진입점
- 연결 상태
- 마지막 동기화
- API 테스트 버튼
- API 오류 메시지

## Instagram 계정 등록

- 플랫폼을 Instagram으로 지정한다.
- 계정 ID와 Access Token을 입력한다.
- API 테스트로 연결 가능 여부를 확인한다.

## Threads 계정 등록

- 플랫폼을 Threads로 지정한다.
- 계정 ID와 Access Token을 입력한다.
- API 테스트로 연결 가능 여부를 확인한다.

## 연결 상태

- 연결됨, 확인 필요, 실패, 토큰 만료 같은 상태를 구분한다.
- 마지막 동기화 시각을 계정별로 보여준다.

## API 테스트

- 계정별로 실행할 수 있어야 한다.
- 성공 시 연결 상태와 마지막 확인 시각을 갱신하는 흐름을 가진다.
- 실패 시 오류 원인과 다음 조치가 보이도록 한다.

## 노출 제한 원칙

- 계정 ID와 Access Token은 메인 탭과 성과 탭에 직접 노출하지 않는다.
- 다른 탭에서는 계정명, 플랫폼, 연결 상태 정도만 보여준다.
- 토큰 값은 평문으로 반복 노출하지 않는다.
## 동기화 관리 방향

- 계정 설정 탭은 API 연결/동기화 관리 화면이다.
- 최근 게시물 조회 결과는 최종적으로 `ContentItem` 자동 등록/업데이트 흐름으로 이어지는 방향을 기본으로 한다.
- 사용자가 매번 “성과를 연결할 콘텐츠”를 선택하는 흐름은 예외 처리나 수동 보정 흐름으로 낮춘다.
- 토큰과 계정 ID는 이 탭에서만 다루며, 상세 디버그 정보는 접기 영역에 둔다.
