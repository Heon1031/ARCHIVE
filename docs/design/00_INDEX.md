# UX 설계 문서 인덱스

이 폴더는 콘텐츠 운영 대시보드의 UX 구조와 사용자 흐름을 정리한다.

## 문서 목록

- `01_UX_OVERVIEW.md`: 전체 UX 방향과 탭별 역할
- `02_MAIN_TAB_UX.md`: 메인 탭의 영역과 운영 흐름
- `03_PERFORMANCE_TAB_UX.md`: 성과 탭의 분석 구조
- `04_ACCOUNT_SETTINGS_UX.md`: 계정 등록/설정 탭의 연결 관리 구조
- `05_GLOBAL_FILTER_UX.md`: 전체/플랫폼/계정 필터의 위치와 동작
- `06_USER_FLOWS.md`: 주요 사용자 흐름
- `07_MANUAL_INPUT_AUX_FLOW.md`: 수기 입력 보조 흐름

## 읽기 원칙

- 화면 구조를 잡을 때는 필요한 탭 문서만 읽는다.
- 필터 동작은 `05_GLOBAL_FILTER_UX.md`를 기준으로 판단한다.
- 수기 입력 위치는 `07_MANUAL_INPUT_AUX_FLOW.md`를 기준으로 제한한다.
