# 글로벌 필터 UX

## 목적

사용자가 전체 계정, 플랫폼별 계정, 개별 계정 기준으로 주요 화면을 전환할 수 있게 한다.

## 필터 종류

- 전체 계정
- Instagram 전체
- Threads 전체
- 개별 Instagram 계정
- 개별 Threads 계정

## 적용 영역

- 메인 탭의 오늘 할 일, 캘린더, 기획 목록, 제작 상태
- 성과 탭의 전체 성과, 플랫폼별 성과, 계정별 성과, 콘텐츠별 성과
- 계정 등록·설정 탭의 계정 목록

## 위치 원칙

- 주요 탭 상단에 일관된 위치로 둔다.
- 선택된 필터가 현재 화면 기준임을 분명히 보여준다.
- 계정 등록·설정 탭에서는 관리 목적에 맞게 계정 목록 필터로 작동한다.

## 동작 원칙

- 전체 계정은 모든 등록 계정을 포함한다.
- 플랫폼 전체는 해당 플랫폼의 모든 계정을 포함한다.
- 개별 계정은 해당 계정 데이터만 보여준다.
- 계정이 없거나 연결 실패 상태면 빈 상태와 조치 안내를 보여준다.

## 주의 사항

- 필터는 토큰이나 계정 ID 상세값을 노출하지 않는다.
- 수기 입력 데이터는 선택 필터와 관련 있는 보조 항목만 보여준다.
- API 실패 데이터는 필터 결과에서 숨기지 않고 상태로 표시한다.
