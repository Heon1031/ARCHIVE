# 사용자 흐름

## 계정 등록 흐름

1. 계정 등록·설정 탭으로 이동한다.
2. 플랫폼을 Instagram 또는 Threads로 선택한다.
3. 계정 ID와 Access Token을 입력한다.
4. API 테스트를 실행한다.
5. 연결 상태와 마지막 동기화 시각을 확인한다.

## 콘텐츠 기획 흐름

1. 메인 탭에서 계정 필터를 선택한다.
2. 콘텐츠 기획 목록에 아이디어를 추가하거나 확인한다.
3. 플랫폼과 대상 계정을 지정한다.
4. 제작 상태로 이동할 준비를 한다.

## 콘텐츠 일정 등록 흐름

1. 메인 탭에서 캘린더를 확인한다.
2. 콘텐츠 기획 항목에 예정일을 지정한다.
3. 업로드 예정 콘텐츠에 반영된 것을 확인한다.
4. 오늘 할 일과 리마인드에 필요한 작업을 표시한다.

## API 연결 확인 흐름

1. 계정 등록·설정 탭에서 계정을 선택한다.
2. API 테스트 버튼을 실행한다.
3. 성공, 실패, 토큰 만료, 권한 부족 상태를 확인한다.
4. 실패 시 메인 탭에는 확인 필요 작업으로만 표시한다.

## 성과 확인 흐름

1. 성과 탭으로 이동한다.
2. 전체 계정, 플랫폼, 개별 계정 필터를 선택한다.
3. 전체 성과와 콘텐츠별 성과를 확인한다.
4. API 데이터 상태를 함께 확인한다.

## API 실패 시 보조 입력 흐름

1. 메인 탭 또는 성과 탭에서 API 실패 상태를 확인한다.
2. 보조 수기 입력 진입점을 선택한다.
3. 필요한 최소 데이터만 입력한다.
4. 해당 데이터는 보조 표시로만 반영된다.
