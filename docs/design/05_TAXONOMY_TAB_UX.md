# 분류 관리 탭 UX

## 목적
- 분류 관리 탭은 게시물별 수정 화면이 아니라 판단 기준 사전 관리 화면이다.
- 메인 추천과 성과 집계에 사용할 관리 키워드, 콘텐츠 형식, 주제 기준을 정리한다.
- 사용자는 사전에 항목을 추가하거나 삭제할 수 있지만, 이 탭에서 콘텐츠를 작성하거나 게시물별 분류를 보정하지 않는다.

## 포함 영역
- 관리 키워드 사전
- 콘텐츠 형식 사전
- 주제 사전
- 제외어/불용어 기준 안내
- 콘텐츠 형식 설명
- 불용어 사전 추가/삭제

## 제외 영역
- 게시물별 수동 보정 UI
- 콘텐츠 작성/추가 폼
- 성과 수치 입력
- API 동기화 관리
- AI 자동 분류 설정

## 저장 방식

- 사전 값은 localStorage의 `content-dashboard.taxonomy`에 저장한다.
- 관리 키워드, 콘텐츠 형식, 주제, 불용어는 각각 추가/삭제할 수 있다.
- `기타`는 기본 분류 fallback이므로 관리 키워드/주제/형식에서 삭제하지 않는다.

## 다른 탭과의 관계
- 메인 탭: 사전 기준을 바탕으로 오늘/이번 주 운영 판단과 날짜별 추천을 보여준다.
- 성과 탭: 사전 기준을 바탕으로 주제/형식/키워드별 성과 근거를 보여준다.
- 계정 설정 탭: API 연결과 게시물 동기화를 담당한다.
