# 데이터 모델 문서 인덱스

이 폴더는 프론트 MVP 구현 전에 필요한 데이터 모델, 저장 정책, API 정규화 기준을 정리한다.

## 문서 목록

- `01_MODEL_OVERVIEW.md`: 핵심 모델 관계와 연결 원칙
- `02_ACCOUNT_MODEL.md`: Account 모델 설계
- `03_CONTENT_ITEM_MODEL.md`: ContentItem 모델 설계
- `04_INSIGHT_RECORD_MODEL.md`: InsightRecord 모델 설계
- `05_FILTER_AND_VIEW_MODEL.md`: 전역 필터와 화면 적용 기준
- `06_LOCAL_STORAGE_POLICY.md`: MVP localStorage 저장 정책
- `07_API_DATA_NORMALIZATION.md`: API 응답 정규화 원칙
- `08_MANUAL_DATA_POLICY.md`: 수기 입력 데이터 정책

## 읽기 원칙

- 계정 구조는 `02_ACCOUNT_MODEL.md`를 기준으로 한다.
- 콘텐츠 구조는 `03_CONTENT_ITEM_MODEL.md`를 기준으로 한다.
- 성과 구조는 `04_INSIGHT_RECORD_MODEL.md`와 `07_API_DATA_NORMALIZATION.md`를 함께 확인한다.
