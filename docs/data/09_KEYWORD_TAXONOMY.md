# 키워드/분류 사전 기준

## 목적
- SNS 게시물 DB와 성과 데이터를 운영 판단에 맞게 해석하기 위한 사전이다.
- 분류 관리 탭은 게시물별 보정이 아니라 관리 키워드, 주제, 콘텐츠 형식, 불용어 사전을 관리한다.
- API로 게시물이 들어오면 이 사전을 기준으로 주제/형식/키워드를 추정한다.

## 저장 원칙
- 사전 값은 `content-dashboard.taxonomy` localStorage key에 저장한다.
- `ContentItem.topicKeywords`는 관리 키워드만 저장한다.
- 접속사, 인사말, 일반 문장 조각은 저장하지 않는다.
- 소파, 우산 같은 단순 사물명은 기본 통계 키워드로 쓰지 않는다.
- 사용자가 사전에 직접 추가한 키워드는 허용한다.
- 성과 수치는 계속 `InsightRecord`에만 저장한다.
