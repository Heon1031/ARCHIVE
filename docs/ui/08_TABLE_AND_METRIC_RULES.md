# 표와 지표 규칙

## 성과 탭 기본 원칙

- 핵심 성과를 먼저 요약하고 세부 데이터는 표나 목록으로 제공한다.
- 차트는 필요한 경우에만 제한적으로 사용한다.
- 성과 화면이 숫자와 그래프로 과밀해지지 않게 한다.

## 지표 카드

- 전체 성과 요약에 사용한다.
- 지표명, 값, 기간, 적용 필터를 함께 보여준다.
- API 데이터와 수기 입력 데이터의 출처를 구분한다.

## 전체 성과

- 선택된 기간과 필터 기준의 합산 지표를 보여준다.
- Instagram과 Threads가 함께 포함되면 플랫폼 구분을 유지한다.

## 플랫폼별 성과

- Instagram 전체와 Threads 전체를 비교한다.
- 플랫폼별로 지원하지 않는 지표는 빈 값처럼 보이지 않게 상태로 표시한다.

## 계정별 성과

- 계정명, 플랫폼, 주요 지표를 행 단위로 보여준다.
- 연결 실패 계정은 성과 없음이 아니라 API 상태 문제로 구분한다.

## 콘텐츠별 성과

- 콘텐츠 제목, 계정, 플랫폼, 날짜, 핵심 지표를 보여준다.
- API 응답이 없는 콘텐츠는 별도 상태로 표시한다.

## 수기 입력 데이터 출처

- 수기 입력 데이터는 보조 출처로 표시한다.
- API 데이터와 같은 신뢰도로 보이지 않게 한다.
- 수기 입력만으로 전체 성과 판단을 강하게 유도하지 않는다.
