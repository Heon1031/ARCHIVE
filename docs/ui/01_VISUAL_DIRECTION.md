# 시각 방향

## 전체 톤

개인 창작자가 매일 쓰는 콘텐츠 운영 도구처럼 차분하고 명확하며 정돈된 화면을 지향한다.

## 지향하는 인상

- 블로그 관리자형 운영 도구
- 반복 작업에 부담이 적은 화면
- 계정, 콘텐츠, 상태를 빠르게 스캔할 수 있는 구조
- 꾸밈보다 정보 우선순위가 분명한 구성

## 정보 표현 원칙

- 중요한 작업과 상태를 먼저 보이게 한다.
- 장식 요소보다 필터, 상태, 날짜, 계정 정보를 우선한다.
- API 연결 상태와 실패 상태는 숨기지 않고 명확히 보여준다.
- 수기 입력은 보조 기능처럼 낮은 시각 우선순위로 둔다.

## 피해야 할 디자인 톤

- 화려한 SaaS 랜딩 페이지 느낌
- 과도한 그라데이션과 장식 카드
- 마케팅용 히어로 섹션
- 지나치게 큰 숫자와 차트 중심 화면
- 수기 입력을 핵심 CTA처럼 보이게 하는 구성
