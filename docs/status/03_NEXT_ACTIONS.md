# 다음 작업 후보

## 우선순위

1. 로컬에서 앱을 실행한다.
2. 실제 테스트용 Instagram / Threads 계정을 등록한다.
3. API 연결 확인을 실행한다.
4. 최근 게시물 조회를 실행한다.
5. 게시물 인사이트 조회를 실행한다.
6. 인사이트를 성과로 저장한다.
7. 성과 탭 반영 여부를 확인한다.
8. 발견한 이슈를 기록한다.

## 테스트 후 판단할 것

- 현재 프론트 단독 MVP를 개인용으로 유지할지 결정한다.
- 배포를 고려한다면 백엔드 프록시와 토큰 저장 구조를 먼저 설계한다.
- API metric 차이가 확인되면 `NormalizedInsight` 정규화 규칙을 보정한다.
- 실제 실패 응답을 기준으로 실패 메시지를 보정한다.
