# 테스트 체크리스트

## 실행

- [ ] `npm install` 후 의존성이 설치되는지 확인
- [ ] `npm run dev`로 앱이 실행되는지 확인
- [ ] `npm run build`가 성공하는지 확인
- [ ] 브라우저에서 로컬 앱 화면이 정상 표시되는지 확인

## 계정 등록/설정

- [ ] Instagram 계정을 등록할 수 있는지 확인
- [ ] Threads 계정을 등록할 수 있는지 확인
- [ ] Brunch / Blog / Other 계정을 등록할 수 있는지 확인
- [ ] 계정 수정이 동작하는지 확인
- [ ] 계정 비활성화 상태가 표시되는지 확인
- [ ] Access Token이 계정 카드에서 원문으로 노출되지 않고 마스킹되는지 확인

## API 흐름

- [ ] 실제 테스트용 계정을 등록하는지 확인
- [ ] Access Token이 없을 때 API 연결 확인이 실패 메시지를 보여주는지 확인
- [ ] 계정 ID가 없을 때 API 연결 확인이 실패 메시지를 보여주는지 확인
- [ ] API 연결 확인 버튼이 계정별로 동작하는지 확인
- [ ] 최근 게시물 조회가 계정별 상세 영역 안에서 동작하는지 확인
- [ ] 게시물 인사이트 조회가 최근 게시물 항목 안에서 동작하는지 확인
- [ ] API 실패 시 앱 전체가 멈추지 않는지 확인

## 성과 저장

- [ ] 인사이트 조회 성공 후 성과 저장 버튼이 표시되는지 확인
- [ ] 저장 전에 연결할 콘텐츠를 선택해야 하는지 확인
- [ ] API 성과 저장 후 성과 탭에 반영되는지 확인
- [ ] 같은 콘텐츠의 API 성과가 중복으로 계속 쌓이지 않는지 확인
- [ ] API 미지원 플랫폼에서 수기 성과 보조 입력이 가능한지 확인
- [ ] 수기 성과 저장 후 성과 탭에 반영되는지 확인

## 유지와 노출

- [ ] 새로고침 후 계정/콘텐츠/성과 기록이 유지되는지 확인
- [ ] 메인 탭에 Access Token 원문이 노출되지 않는지 확인
- [ ] 성과 탭에 Access Token 원문이 노출되지 않는지 확인
- [ ] 브라우저 콘솔에 Access Token 원문이 출력되지 않는지 확인
- [ ] 발견한 이슈를 별도 기록으로 남기는지 확인
