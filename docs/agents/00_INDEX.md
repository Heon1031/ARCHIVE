# AI 에이전트 협업 규약 인덱스

이 폴더는 콘텐츠 운영 대시보드 프로젝트에서 AI 에이전트가 역할별로 필요한 규칙만 읽고 작업하기 위한 문서 모음이다.

## 공통으로 읽을 파일

- 모든 에이전트: `01_GLOBAL_RULES.md`
- 제품 방향 확인이 필요한 경우: `02_PRODUCT_CONTEXT.md`
- 작업 요청 작성 시: `90_TASK_TEMPLATE.md`
- 검토 요청 작성 시: `91_REVIEW_TEMPLATE.md`
- 다른 에이전트에게 넘길 때: `92_HANDOFF_TEMPLATE.md`

## 역할별 파일

- PM / Product Planner: `10_PLANNER_PM.md`
- Frontend Developer: `20_DEV_FRONTEND.md`
- Backend Developer: `21_DEV_BACKEND.md`
- API Integration Developer: `22_DEV_API_INTEGRATION.md`
- UX Structure Designer: `30_DESIGN_UX_STRUCTURE.md`
- UI System Designer: `31_DESIGN_UI_SYSTEM.md`
- Content Strategy / Growth Marketer: `40_MARKETER_CONTENT_STRATEGY.md`
- Content Manager Agent: `41_CONTENT_MANAGER_AGENT.md`
- QA Review Mode: `50_QA_REVIEW.md`

## 읽기 원칙

- 모든 작업에서 전체 문서를 읽지 않는다.
- 담당 역할 파일과 필요한 공통 파일만 읽는다.
- 다른 역할의 판단이 필요하면 해당 역할 파일을 확인하거나 인수인계한다.
- 앱 코드 구현 전에 제품 방향과 금지 사항을 먼저 확인한다.
