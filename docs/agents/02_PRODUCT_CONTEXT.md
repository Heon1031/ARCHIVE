# 제품 맥락

이 프로젝트는 여러 Instagram / Threads 계정을 운영하는 개인 창작자를 위한 콘텐츠 운영 대시보드다.

## 핵심 방향

- Instagram / Threads API 연결을 우선한다.
- 여러 계정을 등록하고 전체 계정 또는 개별 계정 기준으로 볼 수 있어야 한다.
- 수기 입력은 메인 기능이 아니라 보조 기능이다.
- 브런치, 블로그, API 미지원 플랫폼, API 실패 케이스만 수기 입력 대상으로 본다.

## 주요 탭 구조

- 메인 탭: 콘텐츠 캘린더, 기획, 제작 상태, 리마인드, 오늘 할 일
- 성과 탭: 전체 성과, 플랫폼별 성과, 계정별 성과, 콘텐츠별 성과
- 계정 등록/설정 탭: 계정 ID, Access Token, 연결 상태, 마지막 동기화, API 테스트

## 제품 판단 기준

- API 연동 없이 수기 입력 중심으로 흐르지 않게 한다.
- 계정 필터는 전체 계정과 개별 계정 보기를 모두 지원해야 한다.
- 메인 탭은 실행 관리, 성과 탭은 분석, 설정 탭은 연결 관리에 집중한다.
- MVP에서는 핵심 API 연결과 운영 흐름을 먼저 완성한다.
