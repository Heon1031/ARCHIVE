# MVP 제외 범위

MVP에서는 핵심 API 연결과 멀티 계정 콘텐츠 운영 흐름에 집중한다.

## 제외 기능

- 자동 업로드
- 상용 OAuth 전체 플로우
- 로그인
- 결제
- 팀 협업
- Google Calendar 연동
- Google Sheets 연동
- 고급 AI 추천 시스템
- 자동 댓글
- 자동 DM
- 복잡한 승인 워크플로우

## 제외 이유

- MVP의 핵심은 계정 등록, API 연결 상태, 콘텐츠 운영, 성과 확인이다.
- 자동 업로드와 OAuth 전체 플로우는 권한, 보안, 정책 검토 범위가 크다.
- 로그인, 결제, 팀 협업은 개인 창작자용 1차 운영 대시보드의 필수 조건이 아니다.
- 외부 도구 연동과 고급 AI 추천은 API 기반 기본 흐름이 안정화된 뒤 검토한다.

## 판단 기준

- Instagram / Threads API 연결에 직접 필요하지 않으면 후순위로 둔다.
- 메인/성과/계정 등록·설정 탭의 핵심 역할을 흐리면 제외한다.
- 수기 입력이 중심 기능처럼 커지면 MVP 범위에서 줄인다.
