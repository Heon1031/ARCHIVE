# 탭 구조

## 메인 탭

### 목적

오늘 해야 할 콘텐츠 운영 작업과 제작 흐름을 관리한다.

### 포함 요소

- 콘텐츠 캘린더
- 콘텐츠 기획 목록
- 제작 상태
- 오늘 할 일
- 리마인드
- 전체 계정/개별 계정 필터
- API 실패 또는 미지원 채널의 보조 수기 기록 진입점

### 제외 요소

- 상세 성과 분석
- 토큰 관리
- API 권한 설정
- 계정 연결 테스트 중심 화면

## 성과 탭

### 목적

API 기반 성과를 전체/플랫폼별/계정별/콘텐츠별로 확인한다.

### 포함 요소

- 전체 성과 요약
- Instagram / Threads 플랫폼별 성과
- 계정별 성과 비교
- 콘텐츠별 성과
- API 데이터 로딩/오류/빈 상태

### 제외 요소

- 콘텐츠 제작 상태 관리
- 계정 토큰 입력
- 수기 입력 중심 통계
- 자동 업로드 설정

## 계정 등록·설정 탭

### 목적

계정 연결과 API 상태를 관리한다.

### 포함 요소

- 계정 ID
- Access Token 입력 또는 관리 진입점
- 플랫폼 선택
- 연결 상태
- 마지막 동기화
- API 테스트
- API 오류 메시지

### 제외 요소

- 콘텐츠 캘린더
- 콘텐츠 기획 보드
- 성과 상세 분석
- 마케팅 추천 기능
## 탭 역할 재정렬

- 메인 탭은 월간 캘린더가 첫 영역이다.
- API로 가져온 업로드 완료 게시물, 예정 콘텐츠, 보류 콘텐츠를 캘린더에서 구분해 보여주는 방향을 기본으로 한다.
- 성과 탭은 콘텐츠별 반응을 확인하는 보조 분석 화면이다.
- 계정 등록·설정 탭은 API 연결과 동기화 관리에 집중하며, 최근 게시물 조회 결과는 최종적으로 `ContentItem` 자동 등록/업데이트 흐름과 연결한다.
## 분류 관리 탭

### 목적

SNS에서 가져온 게시물 DB의 주제, 콘텐츠 형식, 관리 키워드를 수동으로 보정한다.
이 탭은 콘텐츠를 새로 작성하거나 추가하는 곳이 아니라, 기존 게시물의 분류 기준을 정리하는 관리자 화면이다.

### 포함 요소

- 관리 키워드 기준
- 콘텐츠 주제 기준
- 콘텐츠 형식 기준
- 게시물별 주제/형식/키워드 보정

### 제외 요소

- 콘텐츠 작성 폼
- 자동 업로드 또는 예약 업로드
- AI API 기반 자동 분류
- 성과 수치 직접 입력 중심 UI
## 탭 역할 재정리

- 메인: 오늘/이번 주 운영 판단과 매니저 캘린더를 보여준다.
- 성과: 운영 판단의 근거가 되는 집계/분석 요약을 보여준다.
- 분류 관리: 판단 기준이 되는 키워드, 주제, 콘텐츠 형식 사전만 관리한다.
- 계정 등록/설정: API 연결, 게시물 동기화, 토큰 입력을 관리한다.
