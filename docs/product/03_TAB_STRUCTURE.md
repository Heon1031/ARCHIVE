# 탭 구조

## 메인 탭

### 목적

오늘 해야 할 콘텐츠 운영 작업과 제작 흐름을 관리한다.

### 포함 요소

- 콘텐츠 캘린더
- 콘텐츠 기획 목록
- 제작 상태
- 오늘 할 일
- 리마인드
- 전체 계정/개별 계정 필터
- API 실패 또는 미지원 채널의 보조 수기 기록 진입점

### 제외 요소

- 상세 성과 분석
- 토큰 관리
- API 권한 설정
- 계정 연결 테스트 중심 화면

## 성과 탭

### 목적

API 기반 성과를 전체/플랫폼별/계정별/콘텐츠별로 확인한다.

### 포함 요소

- 전체 성과 요약
- Instagram / Threads 플랫폼별 성과
- 계정별 성과 비교
- 콘텐츠별 성과
- API 데이터 로딩/오류/빈 상태

### 제외 요소

- 콘텐츠 제작 상태 관리
- 계정 토큰 입력
- 수기 입력 중심 통계
- 자동 업로드 설정

## 계정 등록·설정 탭

### 목적

계정 연결과 API 상태를 관리한다.

### 포함 요소

- 계정 ID
- Access Token 입력 또는 관리 진입점
- 플랫폼 선택
- 연결 상태
- 마지막 동기화
- API 테스트
- API 오류 메시지

### 제외 요소

- 콘텐츠 캘린더
- 콘텐츠 기획 보드
- 성과 상세 분석
- 마케팅 추천 기능
