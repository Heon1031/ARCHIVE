# 기능 우선순위

## 1순위: 멀티 계정 등록/관리

여러 Instagram / Threads 계정을 등록하고 계정별 기본 정보를 관리한다.

## 2순위: 계정 필터

전체 계정 보기와 개별 계정 보기를 모든 주요 화면의 기준으로 제공한다.

## 3순위: 콘텐츠 캘린더/기획

계정 기준으로 콘텐츠 일정, 기획, 제작 상태, 오늘 할 일을 확인한다.

## 4순위: API 연결 상태와 API 테스트

계정별 Access Token, 연결 상태, 마지막 동기화, API 테스트 흐름을 만든다.

## 5순위: API 기반 성과 불러오기

Instagram / Threads API 응답을 가져오고 앱에서 사용할 형태로 정리한다.

## 6순위: 성과 탭

전체/플랫폼별/계정별/콘텐츠별 성과를 분리해 보여준다.

## 7순위: API 미지원 채널 수기 입력 보조

브런치, 블로그, API 실패 케이스를 보조 기록으로 다룬다.
