# 제품 구조 문서 인덱스

이 폴더는 콘텐츠 운영 대시보드의 제품 방향, MVP 범위, 탭 구조, 기능 우선순위를 정리한다.

## 문서 목록

- `01_PRODUCT_DIRECTION.md`: 제품 정체성, 대상 사용자, 핵심 문제와 해결 방향
- `02_MVP_SCOPE.md`: MVP에 포함할 기능과 제외할 기능
- `03_TAB_STRUCTURE.md`: 메인/성과/계정 등록·설정 탭의 목적과 구성
- `04_FEATURE_PRIORITY.md`: 기능 구현 우선순위
- `05_OUT_OF_SCOPE.md`: MVP에서 만들지 않을 기능

## 읽기 원칙

- 제품 방향 확인이 필요하면 `01_PRODUCT_DIRECTION.md`를 읽는다.
- 구현 범위 판단이 필요하면 `02_MVP_SCOPE.md`와 `05_OUT_OF_SCOPE.md`를 읽는다.
- 화면 구조 판단이 필요하면 `03_TAB_STRUCTURE.md`를 읽는다.
- 작업 순서 판단이 필요하면 `04_FEATURE_PRIORITY.md`를 읽는다.
